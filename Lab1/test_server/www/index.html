🔍 测试步骤

  • 1. 启动测试服务器：确保Nginx+PHP服务器运行在80端口
  • 2. 启动代理服务器：运行ProxyServerWithCache.exe，监听10240端口
  • 3. 配置浏览器代理：设置HTTP代理为 127.0.0.1:10240
  • 4. 首次访问：点击测试链接，观察代理服务器日志（应显示"未找到缓存"）
  • 5. 刷新页面：再次访问同一页面，观察日志（应显示"找到缓存"和"If-Modified-Since"）
  • 6. 检查响应：如果服务器返回304，代理应使用缓存；如果返回200，代理应更新缓存
  • 7. 查看缓存文件：检查./cache目录，应该有.cache文件生成

✅ 预期结果

  • 首次请求：代理转发请求 → 服务器返回200 + Last-Modified → 代理保存缓存
  • 再次请求：代理添加If-Modified-Since → 服务器返回304 → 代理使用缓存
  • 缓存文件：./cache目录下生成对应的.cache文件
  • 日志输出：代理服务器显示详细的缓存操作日志
